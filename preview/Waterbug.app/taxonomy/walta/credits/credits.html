About

The Waterbug App works best in Australia and is proudly part of The National Waterbug Blitz.

If you are interested in Waterbugs from this part of the world, you might find The Waterbug Book worth a look (Published 2002 by CSIRO Publishing - we made it as well). It has heaps more info and more photos.

The keys we use are available from the ALT project as pdf documents they are free for anyone to use under a Creative Commons License.

In the spirit of open collaboration the source code to the The Waterbug App is available on GitHub It has been licensed under an AGPL-3.0 license, feel free to send us pull requests.
